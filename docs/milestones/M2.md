# Milestone M2 — Neo4j Integration + Concept CRUD (Create/Get)

## Goal
Persist concepts in Neo4j and expose API endpoints to create and retrieve them.

## What I Built
- Neo4j client (`app/db/neo4j.py`)
- Concept models (`app/models/concept.py`)
- Concept service (`app/services/concept_service.py`)
- Concept endpoints (`app/api/concepts.py`)
- Integration test hitting Neo4j
- Automated testing setup with one command (pytest):
    - automatic test discovery
    - clear console output (pass/fail/skip + timing + summary)
    - timestamped log file saved to backend/test_logs/
    - timestamped HTML report saved to backend/test_reports/
    - scalable hooks for future phases (DB fixtures, CI, etc.)

## API
- POST /concepts
- GET /concepts/{concept_id}

## How to Run
cd backend
.\.venv\Scripts\activate
uvicorn app.main:app --reload --port 8000
