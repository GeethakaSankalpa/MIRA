# Milestone M5 — Basic Semantic Search

## Goal
Implement minimal semantic search using embeddings stored in Postgres (JSONB) and cosine similarity ranking.

## What I Built
- Embedding service using SentenceTransformers
- concept_embeddings table in Postgres
- Embeddings stored on concept create/evolve
- GET /search endpoint (cosine similarity ranking)
- Search results logged to query_logs with match metadata
- Deterministic tests using monkeypatch

## Upgrade Path
Replace Postgres embedding scan with Qdrant for scalable vector search.
