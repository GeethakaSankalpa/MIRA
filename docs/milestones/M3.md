# Milestone M3 — Concept Evolution + History

## Goal
Implement concept versioning by creating new nodes for every update and linking versions with EVOLVED_FROM.

## What I Built
- POST /concepts/{concept_id}/evolve
- GET /concepts/{concept_id}/history
- Neo4j constraint for (concept_id, version)
- Tests for evolution + lineage

## Key Rule
Never overwrite a Concept node.
Every change creates a new version node.

## Checkpoint
- Evolve creates v2, v3...
- Old active version becomes deprecated
- History returns versions newest → oldest
